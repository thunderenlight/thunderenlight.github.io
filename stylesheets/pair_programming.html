Pair Programming

What is pair programming?

Pair programing is when two or more people collaborate in code.

Why is pair programming important to DBC?

It's a common a programming practice that encourages efficiency in learning and improving code. This practice also strengthens communication and technical skills among coders.

Compare driver and navigator.

While the driver writes the actual code the navigator steers through the and provides directions towards the task completion.
