Javascript Objects

Javascript Object is a combination of a Ruby class and a Ruby Hash. In keeping in line with object orientated principles, a JS object has properties and methods attached to it. Javascript Objects can be created using literal and constructor notation. Properties are assigned using name value pairs similar to a Ruby hash.

Creating Objects
var hat = {
style: "baseball",
color: "grey",
brand: "Obey"
},

Adding Methods
hat.whatColor = function() {
this.color
};
console.log(hat.whatColor)

Using Prototypes to create Objects:
function Hat(style, color, brand) {
this.style = style;
this.color = color;
this.brand = brand;
}
var myHat = new Hat("fedora", "black", "Goorin Bros.");
