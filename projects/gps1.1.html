What is pair programming?

Pair programing is when two or more people collaborate in code. It is a great way of sharing authorship of code creation. With two people collaborating on the same document in real time whether virtually or one-on-one in person this allows for give-and-take tag team approach to solving and playing in the name of improving on the task at hand. This allows for using communication and mutual understanding within the context code.
